# boilerplate

Single-page app for organizing boilerplate text. Drag and drop snippets into place and download .txt file when you're done.

<div style="display: flex;">
<video src="./demos/demo.mp4"

## Getting Started
```````
git clone
npm install
npm run start:dev
--> localhost:8080
```````

## Testing
````
npm run cypress
````
