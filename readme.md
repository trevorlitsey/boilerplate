# boilerplate

Single-page app for organizing boilerplate text. Drag and drop snippets into place and download .txt file when you're done.

## Getting Started

```
npm install
npm run dev
--> localhost:8080
```

## Unit Tests

```
npm run test
```

## E2E Tests

```
npm run cypress
```
